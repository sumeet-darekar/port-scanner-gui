# port-scanner-gui
port-scanner-gui is tcp and udp port scanning gui tool.

## Installation

clone the repo.  
`git clone https://github.com/sumeet-darekar/port-scanner-gui.git`  

Run it by
`python3 scanner.py` 
OR
Download exe file.
